# AI Infrastructure Vendors

Tracking infrastructure, platform, and tooling companies in the AI ecosystem. These are vendors enabling AI developers, agents, and automation.

---

## Browser Automation & Web Interaction

- **[Browserbase](https://www.browserbase.com/)** — Managed browser infrastructure for web scraping, RPA, and AI web interaction. API-driven headless browser as a service.

- **[Arcade](https://www.arcade.dev/)** — UI automation and human-in-the-loop execution for AI agents. Bridges the gap between pure automation and tasks requiring human judgment.

## Agent Frameworks & Orchestration

- **[Letta](https://www.letta.com/)** — AI agent framework and platform. Handles agent state management, memory, and orchestration for multi-turn conversations and complex workflows.

## Tool Integration & API Management

- **[Composio](https://composio.dev/)** — Agent tooling and API integration layer. Simplifies connecting AI agents to external APIs and tools.

## Code Execution & Sandboxing

- **[e2b](https://e2b.dev/)** — Secure, sandboxed cloud runtime environment for AI code execution. Enables agents to run code safely in isolated containers.

## Observability & Analytics

- **[Langfuse](https://langfuse.com/)** — LLM observability and tracing platform. Monitoring, debugging, and analytics for LLM applications and agents.

## Compute & Serverless

- **[Modal](https://modal.com/)** — Serverless compute platform optimized for AI workloads. Running functions, models, and long-running services in the cloud.

## Memory & Knowledge Management

- **[mem0](https://mem0.ai/)** — Memory management layer for AI agents. Persistent, personalized memory for agents across sessions and interactions.
