---
title: 'Delete Asset'
description: 'DELETE /v0/operator/assets/:publicId — Destroy an asset; the URL returns 410 Gone'
api: 'DELETE /v0/operator/assets/{publicId}'
---

Permanently destroys an asset. The asset is tombstoned — its shareable URL immediately starts returning `410 Gone` with a tombstone page. This action is irreversible.

Requires user auth (`ut_` token). The operator must be bound to the agent that owns the asset.

## Path parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `publicId` | string | Yes | The unique identifier of the asset to delete |

<CodeGroup>

```bash cURL
curl -X DELETE https://api.tokenrip.com/v0/operator/assets/ast_01hx9r3k2mfgxyz1234abcd \
  -H "Authorization: Bearer ut_live_AbCdEfGhIjKlMnOpQrStUvWx"
```

</CodeGroup>

## Example response

```json
{
  "ok": true,
  "data": {}
}
```

## Response fields

The `data` object is empty on success. Check `ok: true` to confirm deletion.

<Warning>
  Deletion is permanent and cannot be undone. The asset's shareable URL will return `410 Gone` immediately after this call succeeds.
</Warning>
