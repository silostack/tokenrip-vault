# __PROJECTS — Project Directory

Strategic and product initiatives beyond the core RebelFi sales pipeline. Each project has its own CLAUDE.md with a brief description and links to key docs.

## Projects

### [Agent CLI](agent-cli/CLAUDE.md)
RebelFi's financial infrastructure for AI agents. Verification before every transaction, yield on every idle dollar, orchestration for every multi-step operation. Transaction Verification SDK shipped Mar 24, 2026. The core product — directly tied to yield deals and SDK adoption in the sales funnel.

### [Tokenrip](tokenrip/CLAUDE.md)
Agentic collaboration platform. Gives agent-produced assets persistent identity, structured agent-to-agent messaging, and shared workspaces. Five-layer architecture from asset routing through agent-native runtime. Currently in first build phase. Also serves as the infrastructure layer for the Intelligence Engine.

### [10x](10x/CLAUDE.md)
AI-native operations roadmap. Three tracks: Build Machine (auto-deploy + plan-to-implementation pipeline), Intelligence Engine (agentic knowledge marketplace built on Tokenrip), and Agent Deployment (social agents on Agent Commune / Multibook). Goal: progress on multiple fronts even when Simon is focused elsewhere.

### [Agentic Economy](agentic-economy/CLAUDE.md)
Strategic research layer. Houses market intelligence, competitive analysis, landscape tracking, and the strategic frameworks that inform Agent CLI and Tokenrip positioning. Originated as a 4-week exploration sprint (Feb–Mar 2026); now a living intelligence repository.
