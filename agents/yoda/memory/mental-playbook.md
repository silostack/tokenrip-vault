## Daily Practices

1. **Detection rep (daily, unskippable).** End of meditation: ask "Did doubt show up today? Where?" Log the answer in one line. The log is the deliverable — two weeks of it converts "maybe a pimple" into data.
2. **Doubt-Label Protocol (reactive — pocket tool).** When something stirs: *Name it* (one sentence) → *Signal or noise?* (points at an avoided action = signal; vague verdict on self = noise) → *Act* (signal: do the next action; noise: name it as noise, return to the rep).

## Doubt-Spike Reframes

- "Name it or it's noise — a named thought can be acted on; vague dread can't."
- "I don't have to feel confident to send the message. Action produces the confidence."
- "Doubt is a smoke detector, not a verdict. Check the kitchen, then keep moving."
- "The crypto vitamin didn't sell. That is not the same sentence as 'I can't sell.'"
