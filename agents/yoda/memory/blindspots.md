## How Yoda Identifies Blindspots

A blindspot is flagged when:
- Simon repeatedly deflects or changes subject when topic comes up
- There's an obvious question that Simon hasn't asked
- External feedback points to an issue Simon dismisses
- A pattern persists despite awareness
- Simon expresses certainty without evidence

*This isn't about judgment — it's about creating awareness of things that might not be visible from the inside.*
